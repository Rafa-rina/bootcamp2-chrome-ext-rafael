Bootcamp Helper — Extensão MV3

Projeto do Bootcamp II: extensão para Chrome com popup, service worker e content script.

Instalação (modo desenvolvedor)

  1. Abrir chrome://extensions no Chrome.
  2. Ativar Developer mode / Modo do desenvolvedor.
  3. Clicar em Load unpacked / Carregar sem compactação e selecionar a pasta do projeto (a raiz com manifest.json).

Links

  • Releases (download .zip)

Permissões usadas

storage

Como usar

Clique no ícone da extensão no Chrome para abrir o popup. Use "Ping background" para testar comunicação com o service worker.
